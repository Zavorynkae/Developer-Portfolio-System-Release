# Developer-Portfolio-System
**技术栈：**  
React · TypeScript · Vite · Tailwind CSS · Markdown · Feature-based Architecture

### 项目简介

独立开发个人开发者内容管理平台，集成博客、项目展示、工具集合及 AI 模块，实现静态内容管理、响应式展示及自动化部署。

### 核心成果

- 独立设计 **Feature-based Architecture**，将 Blog、Portfolio、Tools、AI 等业务拆分为独立 Feature Module，实现模块解耦与可扩展开发。
- 自研 Markdown 内容解析 Pipeline，支持 Front Matter、目录生成、阅读时间统计及静态内容渲染。
- 构建 TypeScript 类型系统与 Service Layer，实现统一接口抽象，提高开发效率与类型安全。
- 基于 **GitHub Actions** 构建自动化 CI/CD 流程，实现代码提交后自动构建与部署。
- 基于 **GitHub Pages** 部署静态网站，并结合 Hash Router 解决 SPA 刷新路由问题。
- 使用 **Tailwind CSS** 构建响应式 UI，兼容桌面端与移动端浏览体验。
